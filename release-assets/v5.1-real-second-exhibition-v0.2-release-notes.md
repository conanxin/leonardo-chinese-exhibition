## Highlights

- The second exhibition v0.2 is now stable-frozen as a public
  release, with the public URL, the production identity, and the
  reproducible QA pipeline all verified end-to-end.
- The v0.2 iteration corrects three long-standing confusions in
  the v0.1 narrative:
  - **Rijksprentenkabinet vs. herbarium** — the Rijksmuseum
    print room (Rijksprentenkabinet, holding the C-09 / C-10
    cyanotypes) is **not** a herbarium; it is a print / drawing /
    photography department.
  - **Cyanotype vs. photogram vs. traditional print** — the
    Anna Atkins C-09 / C-10 works are cyanotypes (a 19th-century
    photographic contact-printing process) and photograms (camera-
    less images made by placing the object itself on the
    sensitised paper). They are not engravings or lithographs.
  - **Public Domain vs. CC0 1.0** — Public Domain is a legal
    status; CC0 1.0 is a legal *tool* by which an institution
    may waive rights. The two are not equivalent; the C-08 Met
    record is double-confirmed (API `isPublicDomain=true` AND
    public-page Public Domain indicator).
- The v0.2 iteration also distinguishes **IIIF Image API** from
  **IIIF Presentation API**: only the Image API is used here; the
  Presentation API /manifest.json for C-10 is known to return
  HTTP 404 (Rijksmuseum, in this verification round), and the
  exhibition explicitly does **not** make any claim that depends
  on Presentation API evidence.
- The 3-minute visitor guide, the four-section narrative
  arc, the deep-block prompts, and the asset usage map have all
  been re-shaped to support the new distinctions.
- The glossary has been expanded from 12 → 14 terms, with
  `cyanotype` and `photogram` newly defined and the
  `herbarium` / `print` / `public-domain` / `IIIF` entries
  clarified.
