## 标题使用建议

| 渠道 | 推荐 | 不推荐 |
|---|---|---|
| 公众号 | A 类（文学性长标题） | C 类（无上下文易被吞） |
| 小红书 | B 类（短 + 数字 + emoji） | A 类（长标题在信息流里撑不开） |
| X 首句 | C 类（短而有钩） | A / B 类（≥ 60 字会切掉） |
| 作品集项目卡片 | D 类（含技术栈） | B 类（数字 + emoji 在作品集不合适） |

— **本地校验**：A-1 与 C-1 在内容上互相呼应（公众号 / X 双发时可在两版之间做"标题对照实验"）。
— **跨渠道一致性**：所有渠道结尾均链向 `https://conanxin.github.io/leonardo-chinese-exhibition/`，确保点击闭环。
