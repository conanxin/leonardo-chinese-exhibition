# 展品图像候选清单（v1.7）

**目的**：用于"达·芬奇的纸上宇宙"中文数字展览的展陈。每个候选条目都对应一个**展区**、一段**中文图注**、一句**alt 文本**，以及明确**状态**。

**当前实时状态来源**：
- Royal Collection Trust：https://www.rct.uk/collection/ （温莎皇家收藏）
- Leonardo//thek@ 平台：https://teche.museogalileo.it/leonardo/

> 注意：以下候选 `confirmed` = 已确认可在公开网页查到的资源（如来自稳定 URL 的图片元数据 / 可访问的水印条目）；`screenshot-needed` = 仍需要在发布前手动下载/截图存证；`svg-generated` = 已生成原创 SVG；`pending` = 待补。

## v1.7 实际行动

1. **本版本（v1.7）保留的"平台外链 + 卡片化"路线**，原因：原始平台与皇家收藏的图像未经明确的可商用授权，不应直接嵌入展览页面。
2. **本版本在视觉上优先使用原创 SVG 图解**完成展览感。
3. **温莎绘图（展区 3）以"展品卡 + 外链候选"形式呈现**，每张卡片配中文标题、英文原题、RCIN 编号、图注、看点、外链。
