# Leonardo Chinese Exhibition v1.0 Release Notes

## 项目定位
基于 Leonardo//thek@ 平台的中文数字展览，聚焦达·芬奇手稿拆散与复原、平台研究工具，而非传统生平介绍。

## v1.0 完成内容
- 完整 8 个展区中文展览
- 深色手稿风格网页（site/index.html + style.css）
- 目录导航 + 锚点
- Open Graph / meta 标签
- 图片 figure + figcaption + source 结构
- 发布文档（DEPLOYMENT.md、RELEASE_NOTES_v1.0.md）

## 主要页面结构
- 序厅 + 8 个展区
- 策展前言
- 3 分钟 / 10 分钟导览稿（exhibition/）
- 社交媒体介绍稿
