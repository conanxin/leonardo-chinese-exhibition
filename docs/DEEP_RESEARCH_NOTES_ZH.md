## 一、可继续深挖的问题（至少 12 个）

### 关于整理史

1. **梅尔齐笔记本的精确清单与归属演变**：弗朗切斯科·梅尔齐继承的笔记本有多少？它们后来如何流入市场、被谁买走、最终落到哪些馆藏？有没有一份按时间排的流转记录？
2. **莱奥尼裁切的具体边界**：蓬佩奥·莱奥尼裁切装订《大西洋手稿》时，对每一页做了多大程度的修整？有没有任何一份纸页被裁掉过边缘？这些被裁掉的边缘，今天在哪里？
3. **温莎收藏与米兰馆藏的拆分原则**：19 世纪末达·芬奇手稿被分到温莎与米兰两个馆藏时，按什么标准切分？是按尺寸、年代、主题，还是按拍卖批次？
4. **整理者的命名习惯**：17–19 世纪的整理者通常如何命名手稿？「Codex Atlanticus」这个名称什么时候第一次出现？是否经过多次改名？

### 关于材料证据

5. **水印数据库的覆盖范围**：今天常用的达·芬奇手稿水印数据库（Briquet、Piccard 等）覆盖了达·芬奇活动时期意大利北部造纸厂的多少比例？有没有哪个时期的水印数据库明显缺失？
6. **裁切边的判定标准**：什么样的边缘算「被裁过」、什么样的算「原始不齐」？有没有公开的判读图谱？
7. **馆藏编号系统的演变**：温莎 RCIN 编号、安布罗西安 Codex Atlanticus folio 编号，是什么时候、由谁、为什么确定的？编号改变过吗？

### 关于图像与思考

8. **图式与文字的边界**：达·芬奇的「图—文混合」中，哪些图明确是「先画后注」、哪些是「先注后画」、哪些是「同时进行」？有没有办法在纸面上辨认？
9. **跨域图式的来源**：把水流当成血液、把肌肉当成滑轮——这种「跨域图式」是达·芬奇自己的发明，还是从他读过的前辈（维特鲁威、阿尔哈曾）那里继承？
10. **未完成与放弃的痕迹**：手稿里有大量未完成的图、涂掉的论证、空白页——这些「没做的事」能不能被研究？它们在方法论上意味着什么？

### 关于平台与方法

11. **Leonardo//thek@ 的版本演进**：平台从 1.0 到 2.0 主要改了什么？工具边界有没有公开的 changelog？哪些功能是新增的、哪些被替代？
12. **并置比较（Comparative Study）的方法限制**：当两页并排显示时，研究者能不能看到背面？能不能看到透光细节？这些限制是否被工具补足？
13. **复原（Recompositions）的工作流**：平台的「复原」功能目前覆盖了哪些著名页？哪些复原在学界有争议？

## 二、需要后续查证的事实（至少 5 个）

下列陈述在 v2.7 / v2.8 内容中出现过，但在本轮没有逐一核对到一手来源。**未来 round 在引用为确定事实前必须先核验**：

1. **纸页具体年代**：每一页的精确年代判断目前主要依据水印与风格学，不是直接纪年材料。当展览或策展长文中说「约 1490 年」「15 世纪 80 年代」时，是基于哪一份研究文献？
2. **RCIN 与图片对应关系**：温莎皇家收藏的 RCIN 编号与本展览引用的图之间，是否每一张都能一一对应？有没有 RCIN 编号与平台图像 ID 之间的官方映射？
3. **Codex Atlanticus folio 编号细节**：folio 编号（如「Codex Atlanticus, f. 149r」「f. 391ra」）在不同版本的图录里是否完全一致？verso / recto 标记是否总是按同一约定？
4. **平台模块的最新功能边界**：Foliations / Subject Indexes / Photographic Plates / Lexicon / Watermarks / Recompositions / Bibliography / Advanced Search / Comparative Study 九大模块，今天的可访问性、是否需要登录、是否有 API，都是本轮未逐一查证的。
5. **图片授权与再使用范围**：本展览目前对所有 RCIN / Codex Atlanticus 图源笼统说「公共域授权 / Wikimedia Commons 引用」，但每一张图的版权状态与归属要求应该单独查证。**这正是 v2.9 source rights audit 的任务**，不要在 v2.8 假装已经覆盖。

### 补充：写作语气问题

6. **「达·芬奇很聪明 / 他什么都懂」**：这是反话，不是事实。本展览的语气必须保持克制，避免「天才崇拜」式的总结句。
