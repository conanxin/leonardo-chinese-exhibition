# 一件作品的旅程 · 深度研究笔记

> 本文件为 v3.1 second exhibition pilot 的深度研究笔记中文版。
> 用途：演示模板研究笔记模块；不参与 live。

## 假设与边界

- 本 pilot 不引用任何具体作品、具体机构、具体研究者。
- 笔记讨论的是「单件作品研究」的通用方法论，不指向任何特定历史事件。
- 笔记所引用的「典型流程」「常见模型」均来自通用策展实践，不指向具体文献。

## 单件作品研究的四步模型

```
材料 → 流转 → 阅读 → 展示
```

- **材料**：纸张、墨色、折痕、印章、装裱形式、装订方式。
- **流转**：收藏链、修复史、出版史、数字化史、再展览史。
- **阅读**：研究方法、注释传统、材料检测、归属判断。
- **展示**：光线、动线、说明文字、展柜高度、停留时间、拍照政策。

这四步在 pilot 中被映射为 4 个 section，让模型既可看、也可被引用为骨架。

## 源出处链

```
L1 原作机构 / 收藏方
 ↓
L2 数字档案 / 公共域镜像
 ↓
L3 展览截图 / 项目自绘
 ↓
L4 展览展签 / 说明文字
```

每一层都对应 sourceNote + creditLine 双字段。pilot 中 L3 全部为 project-generated。

## 观看动线与停留时间

| 时长 | 行为 | 设计含义 |
|---|---|---|
| 5s | 路过 | 展柜长度、节奏、整体感 |
| 30s | 停留 | 单件展品的标准阅读时长 |
| 90s | 深读 | 研究者模式，需要 detail view / 注释放大 |

pilot 中以 5/30/90 秒作为典型取值，下游展览可基于具体作品 / 观众调研数据重新标定。

## 与具体作品的距离

本 pilot 故意保持与具体作品的距离，目的是：

1. 让模板在「最小可运行实例」下被验证。
2. 避免将模板的成败与具体作品的内容 / 权利状态绑定。
3. 给未来展览项目留出「可填充」的空间。

## 待补研究

- **停留时间标定**：5/30/90 秒是基于通用观众行为观察的取值，下游项目可基于具体作品与观众调研重新标定。
- **策展四角模型**：本 pilot 用 `object-journey.svg` 充当占位图；正式展览中应替换为专属策展四角模型图。
- **正文 200–600 字**：本 pilot 的 section body 由作者撰写，未参考具体作品。下游展览项目应基于具体作品材料撰写。

## 不在本 pilot 范围

- 具体作品 / 机构 / 研究者
- 真实馆藏图 / 机构截图
- 源出处、权利、授权细节
- 双语 / i18n
- 服务端 / 数据库
- SEO / meta-tag 优化
