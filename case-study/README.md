# case-study/ 目录导览

本目录收录**作品集展示**类文档——与 `posts/`（传播材料）分开，专门给雇主、合作方、学术同行、项目评委等 review 用途。

---

## 文档清单

| 文件 | 内容 | 推荐读者 |
|---|---|---|
| `portfolio-case-study.md` | 中文长篇完整案例（14 节）| 5 分钟看的完整读者 |
| `project-onepager.md` | 一页式项目说明 | 5 秒扫描的访客 |
| `portfolio-case-en.md` | 英文版短案例（适合国际作品集）| 国际合作方 / 英文读者 |
| `readme-showcase-section.md` | README 顶部展示段（可直接粘贴）| 项目主页或 README |
| `launch-x-post.md` | X 14 条中文长帖 | X / Twitter 关注者 |
| `project-retrospective.md` | 项目复盘 | 自我复盘 / 内部参考 |
| `README.md` | 本目录导览 | 索引读者 |

---

## 推荐阅读顺序

1. **5 秒**：看 `project-onepager.md` 的"亮点 5 条"段
2. **30 秒**：扫 `portfolio-case-en.md` 或 `readme-showcase-section.md`
3. **5 分钟**：通读 `portfolio-case-study.md`
4. **15 分钟**：再读 `project-retrospective.md`

---

## 与 posts/ 的区别

| 维度 | posts/ | case-study/ |
|---|---|---|
| 内容类型 | 散文 / 脚本 / 标题 / 卡片文案 | 案例文档 |
| 受众 | 普通读者、传播受众 | 雇主、合作方、学术同行、评委 |
| 用途 | 公众号 / X / 小红书 / 视频 / 作品集展示 | 作品集首页 / 简历附件 / 项目提案 |
| 时间长度 | 短（200-2000 字）| 中长（2000-6000 字）|
| 风格 | 感性 + 互动钩子 | 结构化 + 可引用 |
